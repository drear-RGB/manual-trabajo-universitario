# 📘 Manual de Buenas Prácticas para Trabajos Universitarios en Grupo

> Repositorio creado como parte de un proyecto universitario para aprender Git y GitHub como herramienta de colaboración y control de versiones.

Autor: Andrés Bermudez.

## 📌 Descripción

Este repositorio contiene un **manual colaborativo** con buenas prácticas para realizar trabajos universitarios en grupo. Cubre temas como organización del equipo, gestión del tiempo, citación APA, herramientas colaborativas y uso de GitHub.

## 🤝 Cómo contribuir

1. Crea una rama con el nombre de tu sección:
   ```bash
   git checkout -b seccion-tu-nombre
   ```

2. Realiza tus cambios y guárdalos:
   ```bash
   git add .
   git commit -m "Descripción clara de lo que hiciste"
   ```

3. Sube tu rama:
   ```bash
   git push origin seccion-tu-nombre
   ```

4. Abre un **Pull Request** desde GitHub y solicita revisión.

5. Espera la aprobación antes de fusionar con `main`.

---

## 📋 Reglas de contribución

- Cada integrante trabaja en **su propia rama**, nunca directamente en `main`.
- Los mensajes de commit deben ser **descriptivos y en español**.
- Todo Pull Request debe ser **revisado por al menos un compañero** antes de aprobar.
- Los conflictos se resuelven en conjunto.
